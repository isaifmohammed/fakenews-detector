# 🔍 TruthWatch - AI News Integrity Monitor

🌐 Live Demo: https://truthwatch.onrender.com

A real-time fake news detection web app powered by Groq LLM (Llama 3.1) and Flask.

![Flask](https://img.shields.io/badge/Flask-Web%20App-green)
![AI](https://img.shields.io/badge/AI-Groq%20LLM-orange)
![Status](https://img.shields.io/badge/Status-Active-brightgreen)

## 🚀 Features
- 🔴 Detects fake news using Groq LLM (Llama 3.1)
- 📰 Live breaking news feed powered by NewsAPI
- 🌐 Professional news channel style UI
- 🔗 Analyze by URL or plain text
- ⚡ Real-time breaking news ticker

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **AI:** Groq API (Llama 3.1)
- **Frontend:** HTML, CSS, JavaScript
- **APIs:** NewsAPI, Groq API

## ⚙️ Installation
```bash
# Clone the repo
git clone https://github.com/isaifmohammed/fakenews-detector.git

# Install dependencies
pip install -r requirements.txt

# Add API keys to .env file
NEWS_API_KEY=your_newsapi_key
GROQ_API_KEY=your_groq_key

# Run the app
python app.py
```

Open browser at http://127.0.0.1:5000
